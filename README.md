## Getting Started

### Clone the Repository

First, clone the repository to your local machine:

```bash
git clone https://github.com/hxdy-1/FlowLaunch-Assignment.git
cd flowlaunch
```

Install dependency.

```bash
npm install
```

Run locally.

```bash
npm run dev
```
